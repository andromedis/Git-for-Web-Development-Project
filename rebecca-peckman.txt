Part 2 Questions

    1. What is Semantic HTML?

Semantic HTML is HTML that both formats content within tags and whose tag names have meanings that describe the underlying structure of the content.
 
    2. What is HTML used for?

HTML is used to add formatting and structure to a webpage so that it will be displayed a certain and consistent way within any particular web browser.

    3. What is an attribute and where do we put it? 

An attribute is a modifier to a particular HTML tag that gives that tag more information. It is placed within the <> brackets, after the tag name and a space, or after a previous attribute's name and a space.

    4. What is the h1 tag used for? How many times should I use it on a page?

The h1 tag is used to specify the largest heading within the page, or the title of the page's content. It should only be used once in any page.

    5. Name two tags that have required attributes

Two tags that have required attributes are the <a> anchor tag, and the <img> image tag. The <a> tag requires the href attribute used to create a link. The <img> tag requires the src and alt attributes, describing the path to the image intended for display, and text that will be displayed if the image cannot be.

    7. What is an id?

An id is a unique identifier that is assigned to a single html element within it's tag.

    8. What elements can I add an id to? 

The id attribute is a global attribute, and can be added to any element.

    9. How many times can I use the same id on a page?

Any specific id can only be used once on a page.

    10. What is a class? 

A class is an identifier for a particular type of HTML element that can be used over many different elements that need to be grouped and displayed similarly.

    11. What elements can I add a class to? 

The class attribute is a global attribute, and can be added to any element.

    12. How many times can I use the same class on a page? 

The same class can be used an infinite number of times on a page.

    13. How do I get my link to open in a new tab?

You can force the <a> tag to open its in a new tab by adding the target="_blank" attribute to the tag.

    14. What is the alt attribute in the image tag used for? 

The alt attribute in the <img> tag is used to add text that will be displayed if the image cannot be displayed for any reason.

    15. How do I reference an id?

An id is referenced with a # followed by the id name: #[id-name]

    17. What is CSS used for? 

CSS is used to take the content that has been sectioned and structured with HTML and format the aesthetics of how it will be displayed.

    18. How to we select an element? Example - every h2 on the page

An element is selected with its name, followed by a space and brackets containing the property:value pairs of its styling:
h2 {
  color: green;
}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one

A class can be used for any number of elements in a page, while an id can only be used for one. A class might be added to elements that are all images that should be displayed the same way. An id should be added to a single element that needs to be specifically referenced, such as a location name that will be used in Javascript code on the page.

    20. How do we select classes in CSS?

Classes in CSS are selected by a period followed by the class name: .[classname]
